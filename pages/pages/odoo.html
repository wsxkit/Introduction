Odoo Barcode-like 无输入框监听测试页

这个页面模拟 Odoo Barcode 这类 Web 应用常见的“无输入框、监听全局键盘事件”的方式。 页面不会使用文本输入框接收条码，而是监听 document/window 上的 keydown / keypress / keyup / beforeinput / input / paste。
测试重点：如果扫描正文只能通过 Android KeyMultiple / chars 进入输入框，那么在这种无输入框页面里可能无法被 JavaScript 捕获为逐字符扫描。

不要点击任何输入框。本区域只是可聚焦容器，不是 input/textarea。
建议分别测试：无后缀、Enter 后缀、Tab 后缀。
